Local Docker Launcher

先登录，再启动后端。

这个页面不再直接请求你的 docker 后端，而是先把邮箱、密码、OTP 密钥提交给本地启动器。 点击“启动后端并进入聊天”后，本地服务会自动执行 docker build/run，等模型可用后再进入聊天页。

第一步 输入邮箱、密码、OTP 密钥和代理地址。
第二步 点击启动，本地服务去拉起 docker 容器。
第三步 之后每次发消息，都由本地服务代理一次后端请求。
